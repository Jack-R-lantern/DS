## **구현 명세**
> 추상데이터 타입 T\
> 각 메소드의 반환형은 구현언어의 명세에 따름
>### **push(T value)**
> 스택에 요소 삽입.
>### **pop()**
> 스택의 top을 제거.
>### **top()**
> 스택의 top을 확인.
>### **empty()**
> 스택이 비어있는지 확인.
